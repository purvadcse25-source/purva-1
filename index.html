Online Tutorial

Explanation

HTML is used to create web pages. It uses tags like headings and paragraphs.

  • <h1> - Heading
  • <p> - Paragraph

Code Playground

Sample Code:

<html>
<body>
<h1>Hello</h1>
<p>This is tutorial</p>
</body>
</html>

Output

The above code will display a heading and a paragraph on the webpage.
